# 2026-04-17 12:25 - 产品详情页完成

## 目标
解决 Next.js 16 `output: export` 下 `/products/[id]` 动态路由 `generateStaticParams` 报错问题。

## 方案
将动态路由 `/products/[id]/page.tsx` 改为静态路由 `/product/page.tsx` + query 参数：
- URL: `/product?id=<product_id>`
- `page.tsx`: 纯静态 Server Component（无需 generateStaticParams）
- `ProductDetailClient.tsx`: `'use client'`，用 `useSearchParams()` 读取 `id`，`useEffect` 动态加载产品数据，`Suspense` 包裹处理路由加载状态
- `products/page.tsx`: 两处跳转链接从 `/products/${product.id}` 改为 `/product?id=${product.id}`
- 相关推荐链接同步更新为 query params 格式
