# 丽姿秀美容院预约管理系统

基于 Next.js + TypeScript + Tailwind CSS + Supabase 构建。

## 功能
- 在线预约（选择服务/美容师/时间，冲突检测）
- 产品商城（购物车、结算、收款码支付）
- 商家后台（订单管理、产品管理、数据分析）
- 日历视图（查看每日预约安排）
- 客户管理

## 技术栈
- 前端：Next.js 16 + React + TypeScript + Tailwind CSS
- 后端：Next.js API Routes
- 数据库：Supabase (PostgreSQL)
- 图表：Recharts
- 部署：Vercel

## 本地开发
```bash
npm install
npm run dev
```
