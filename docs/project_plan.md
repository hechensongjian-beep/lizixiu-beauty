# 丽姿秀美容院网站项目计划

## 项目概述
构建一个现代化、高端大气的美容院网站，实现预约制、商品购买、用户通知、线上支付、客服对话等功能。

## 核心功能需求
1. **预约制功能**
   - 客户前端预约：选择日期、时间、项目
   - 后台商家查看：日程表视图，清晰显示预约时间段
   - 时间冲突检测：已预约时间段对客户不可选
   - 商家可删除预约：避免顾客未取消影响其他预约
   - 项目时长计算：自动占用相应时间段（如120分钟项目占用12:00-14:00）

2. **商品商城**
   - 商家上传商品：图片、基础信息、价格
   - 前端商城展示：新品推荐、商品分类
   - 购物车和购买流程

3. **用户系统**
   - 用户注册/登录
   - 用户管理（个人信息、预约历史、订单记录）

4. **通知系统**
   - 新品上市、活动促销等消息推送给注册用户

5. **线上支付**
   - 集成支付网关（如支付宝、微信支付）

6. **客服对话**
   - 后台与客户实时对话（如在线聊天）

7. **后台管理**
   - 简单易用的操作界面，适合非技术人员
   - 上传商品、管理预约、查看订单等

## 技术栈选择
- **前端**：Next.js (React) + TypeScript + Tailwind CSS
- **后端**：Next.js API Routes (Node.js)
- **数据库**：Supabase (PostgreSQL) 或 SQLite
- **认证**：Supabase Auth 或 NextAuth.js
- **支付**：Stripe 或国内支付接口（需申请）
- **部署**：Vercel (前端) + Supabase (后端)

1. **环境搭建**：安装 Node.js、初始化项目、安装依赖

3. **基础框架**：创建布局、导航栏、页面结构
4. **预约功能**：实现日历组件、预约逻辑、后台管理
5. **商品商城**：商品展示、购物车、下单流程
6. **用户系统**：注册登录、个人中心
7. **支付集成**：支付接口对接
8. **通知系统**：消息推送
9. **客服对话**：实时聊天界面
10. **测试部署**：功能测试、性能优化、部署上线
